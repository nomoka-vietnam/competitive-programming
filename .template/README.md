# Template

Generic C++ template for competitive programming solutions.

**File:** `20260617.cpp`

### Features

| Macro         | Description                  |
|---------------|------------------------------|
| `OPTIMIZE`    | Fast I/O sync and untie      |
| `FILESTD(x)`  | Redirect I/O to `x.inp/x.out`|
| `FOR(a,b,c)`  | Inclusive forward loop       |
| `REV(a,b,c)`  | Inclusive reverse loop       |

### Customization

Edit the template file before copying to add problem-specific includes or constants.
