# Competitive Programming

Solutions to competitive programming problems, organized by platform.

## Platforms

- [LQDOJ](https://lqdoj.edu.vn/) — Vietnamese online judge for high school students.

## Template

A generic C++ template is available under `.template/`. It includes:
- Common type aliases (`ll`, `ull`, `vi`, `pii`, `mii`)
- Loop macros (`FOR`, `REV`)
- Utility macros (`pb`, `all`, `size`)
- Fast I/O optimization (`OPTIMIZE`)
- File-based I/O helper (`FILESTD`)

## Usage

```bash
# Clone the repository
git clone https://github.com/nomoka-vietnam/competitive-programming.git

# Copy the template for a new problem
cp .template/20260617.cpp path/to/problem.cpp
```
